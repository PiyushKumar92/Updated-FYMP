Privacy & Security

Your data protection and privacy are our top priorities

Data Protection

Encryption
  • All data encrypted in transit using HTTPS/TLS
  • Database encryption at rest
  • Secure file storage with access controls
  • Password hashing using industry standards
Access Control
  • Role-based access permissions
  • Admin audit trails and logging
  • Secure authentication system
  • Session management and timeouts

Privacy Policy

Information We Collect

We collect only the information necessary to provide our missing person search services:

  • Account Information: Username, email address, and encrypted password
  • Case Data: Photos, videos, and details of missing persons (with consent)
  • Usage Data: Platform usage statistics for service improvement
  • Technical Data: IP addresses, browser information for security purposes
How We Use Your Information
  • Process missing person cases using AI facial recognition
  • Provide search results and notifications
  • Improve our AI algorithms and platform performance
  • Ensure platform security and prevent misuse
  • Communicate important updates and support
Data Sharing

We never sell or share your personal data with third parties for commercial purposes. Data may only be shared:

  • With law enforcement agencies when legally required
  • With your explicit consent for specific cases
  • In anonymized form for research and development

Technical Security

  • Secure Database
    SQLAlchemy ORM with parameterized queries preventing SQL injection
  • Input Validation
    Comprehensive file validation and sanitization for uploads
  • Background Processing
    Secure Celery task queue with Redis for AI processing

User Rights

  • Data Access
    View and download all your personal data anytime
  • Data Correction
    Update or correct your information through your profile
  • Data Deletion
    Request complete account and data deletion
