Getting Started

Creating an account is simple:

  1. Click the "Register" button in the top navigation
  2. Fill in your username, email, and password
  3. Verify your email address (if required)
  4. Start creating missing person cases immediately

Registration is completely free and takes less than 2 minutes.

After logging in, follow these steps:

  1. Click "New Case" in the navigation or dashboard
  2. Fill in the missing person's details (name, age, description)
  3. Upload clear, recent photos of the person
  4. Optionally upload search videos (surveillance footage, etc.)
  5. Submit the case for AI processing

Our AI will start processing immediately and notify you of any matches.

AI Technology

Our AI system uses advanced computer vision technology:

  • Face Detection: Uses HOG and CNN models to locate faces in images/videos
  • Feature Extraction: Creates unique facial encodings using deep learning
  • Similarity Matching: Compares encodings to find potential matches
  • Confidence Scoring: Provides percentage confidence for each match

The system achieves 95%+ accuracy in controlled conditions.

Processing times depend on the content:

  • Photos only: 1-2 minutes
  • Short videos (< 5 min): 5-10 minutes
  • Long videos (> 10 min): 15-30 minutes

You'll receive real-time notifications as matches are found. The system processes in the background using Celery task queues.

Confidence scores indicate match likelihood:

  • 90%+: Very high confidence - likely match
  • 80-89%: High confidence - strong possibility
  • 70-79%: Medium confidence - worth investigating
  • 60-69%: Low confidence - possible but uncertain
  • < 60%: Very low confidence - unlikely match

Always verify matches manually, especially for important decisions.

Privacy & Security

Yes, we implement enterprise-grade security:

  • All data encrypted in transit (HTTPS/TLS)
  • Database encryption at rest
  • Secure file storage with access controls
  • Regular security audits and updates
  • Role-based access permissions

View our Privacy & Security page for complete details.

Case visibility is controlled:

  • You: Full access to your cases and results
  • Admins: Can view cases for support and moderation
  • Public Directory: Basic case info (name, photo) for active cases only
  • Other Users: Cannot access your private case details

You can withdraw cases anytime to remove them from public view.

Technical Support

Supported Image Formats:

  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • GIF (.gif)
  • BMP (.bmp)
  • WebP (.webp)

Supported Video Formats:

  • MP4 (.mp4)
  • AVI (.avi)
  • MOV (.mov)
  • MKV (.mkv)
  • WebM (.webm)

Maximum file size: 16MB for images, 100MB for videos.
